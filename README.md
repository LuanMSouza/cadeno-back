## 📋 Sobre

Servidor Node.js com endpoints REST para o frontend do Cadeno, com regras de negócio e persistência de dados.

## 🚀 Como Rodar

**Pré-requisitos:** Node.js 18+

```bash
git clone https://github.com/LuanMSouza/cadeno-back
cd cadeno-back
npm install
cp .env.example .env
npm run dev
```

API disponível em `http://localhost:3001`.
